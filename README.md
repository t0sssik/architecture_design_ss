# Разработка программного модуля автоматической оценки пространственного разрешения аэрофотоснимков
## Актуальность
Исследование сосредоточено на современных системах, которые собирают и используют аэрофотоснимки для решения прикладных задач в картографии, городском планировании и сельском хозяйстве. Однако на практике метаданные изображений зачастую отсутствуют, повреждены или содержат неточные сведения. Одним из таких параметров является пространственное разрешение (Ground Sample Distance, GSD), которое может иметь значительную погрешность из-за особенностей используемой аппаратуры. С одной стороны, барометрические альтиметры являются широко используемым и доступным инструментом для определения высоты, позволяющей вычислить значение GSD, а с другой стороны – они подвержены существенным ошибкам из-за изменения атмосферного давления и не учитывают особенности рельефа поверхности. Появляется актуальный вопрос: как добиться точных и надежных измерений высоты, которые были бы независимы от атмосферных условий и учитывали реальный рельеф местности?
## Цель
Разработка программного модуля автоматического определения пространственного разрешения аэрофотоснимков урбанизированных и природных ландшафтов с использованием методов глубокого обучения
## Задачи
1.	Выполнить анализ современных методов оценки пространственного разрешения аэрофотоснимков:
1.1.	Провести обзор научной литературы и существующих систем для оценки GSD.
1.2.	Выявить и проанализировать недостатки традиционных методов, основанных на данных барометрических альтиметров.
2.	Провести проектирование программного модуля и модели глубокого обучения:
2.1.	Спроектировать архитектуру программного модуля для автоматической оценки GSD.
2.2.	Спроектировать модели глубокого обучения.
3.	Реализовать спроектированное решение.
4.	Провести экспериментальное исследование точности разработанных моделей на тестовых данных.
## Описание исследования (ключевые этапы)
Мы рассмотрели различные нейросетевые подходы для задачи оценки пространственного разрешения по снимкам дистанционного зондирования урбанизированных и природных ландшафтов. Среди них: использование признаковых карт с разных уровней энкодера, предварительное обучение энкодера на задаче восстановления изображения и использование относительной функции потерь. В качестве ключевой метрики рассматривается средняя абсолютная относительная ошибка, показывающая сопоставимые оценки предсказаний на диапазоне. Все рассматриваемые модели обучались на снимках разных типов местности. Также для их обучения применялись аугментации масштабирования для расширения диапазона GSD в выборках данных.
## Основные выводы
В качестве базовой модели была взята ResNet18, которая на имеющихся данных демонстрирует относительную ошибку в диапазоне от 2.73% до 15.6%. Применение относительной функции потерь непосредственно при обучении позволяет улучшить показатели на данных с низким пространственным разрешением с 15.6% до 14.7%. В то же время использование подхода SSL незначительно улучшает показатели на высоких значениях GSD. Комбинация SSL с FPN [5] достигает 3.61% на наборе со смешанным типом местности. Полученные результаты позволяют использовать обученные модели для решения прикладных задач. Однако для каждого конкретного случая необходимо выбирать наиболее подходящий метод.
